# Alert ribbon

## Types

Set a alert type for color coded alerts. Please note that inline links can only be in the end of the AlertRibbon in Figma/Design Library so this code might be utpdated to that.

<div>
  <AlertRibbon header="Info heading." isCloseable={false}>
    AlertRibbon content placed inside a paragraph. Inline links,{' '}
    <a href="#">info</a>, will inherit color from alert to make sure contrast is
    applied.
  </AlertRibbon>
  <AlertRibbon type="success" header="Success heading." isCloseable={false}>
    AlertRibbon content placed inside a paragraph. Inline links,{' '}
    <a href="#">success</a>, will inherit color from alert to make sure contrast
    is applied.
  </AlertRibbon>
  <AlertRibbon type="warning" header="Warning heading." isCloseable={false}>
    AlertRibbon content placed inside a paragraph. Inline links,{' '}
    <a href="#">warning</a>, will inherit color from alert to make sure contrast
    is applied, like this .
  </AlertRibbon>
  <AlertRibbon type="danger" header="Danger heading." isCloseable={false}>
    AlertRibbon content placed inside a paragraph. Inline links,{' '}
    <a href="#">danger</a>, will inherit color from alert to make sure contrast
    is applied, like this .
  </AlertRibbon>
</div>

## AlertRibbon with Footer Buttons

Button components can be passed in as a footer

<div>
  <AlertRibbon header="Info" footer={<FooterInfo />} isCloseable={true}>
    This is an alert type <a href="#">info with a link</a>.
  </AlertRibbon>
  <AlertRibbon
    type="success"
    header="Success"
    footer={<FooterSuccess />}
    isCloseable={true}
  >
    This is an alert type <a href="#">success with a link</a>.
  </AlertRibbon>
  <AlertRibbon
    type="warning"
    header="Warning"
    footer={<FooterWarning />}
    isCloseable={true}
  >
    This is an alert type <a href="#">warning with a link</a>.
  </AlertRibbon>
  <AlertRibbon
    type="danger"
    header="Danger"
    footer={<FooterDanger />}
    isCloseable={true}
  >
    This is an alert type <a href="#">danger with a link</a>.
  </AlertRibbon>
</div>
